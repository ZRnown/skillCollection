# First-Time Setup

Create one of these files:

- Project-level: `.thy1cc-skills/thy1cc-post-to-qiehao/EXTEND.md`
- User-level: `$HOME/.thy1cc-skills/thy1cc-post-to-qiehao/EXTEND.md`

Recommended template:

```md
chrome_profile_path: $HOME/.local/share/qiehao-browser-profile
editor_url: https://om.qq.com/article/articlePublish
```

Notes:

- `chrome_profile_path`: point this to the Chrome profile that already holds your Qiehao login state.
- `editor_url`: keep the official article route here; if the platform changes paths later, override it here instead of editing the script.
